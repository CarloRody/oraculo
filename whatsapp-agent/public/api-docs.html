📱 API de WhatsApp

Envio automatizado de mensagens WhatsApp via API, com cobrança por mensagem descontada do seu saldo. Veja também o painel de administração.

Autenticação

Todas as chamadas usam a mesma chave de acesso já utilizada nas demais APIs do Oráculo (ex: /api/chat), enviada no cabeçalho:

X-Oraculo-Key: SUA_CHAVE_DE_ACESSO

Sua chave de acesso é fornecida pelo administrador do Oráculo. Cada requisição é identificada e cobrada na conta dessa chave.

Base URL: este endpoint fica no serviço principal do Oráculo (não no serviço de WhatsApp) — use o mesmo endereço que você já usa pra /api/chat, por exemplo https://SEU-DOMINIO-ORACULO.

Enviar mensagem

POST/api/whatsapp/send

Envia uma mensagem de texto pelo número de WhatsApp vinculado à área informada. A conexão usada é resolvida automaticamente a partir do vínculo área ↔ conexão configurado no seu cadastro (feito pelo administrador). Cada envio bem-sucedido é cobrado e descontado do seu saldo, conforme o preço configurado no seu plano para essa área.

Parâmetros do corpo (JSON)

CampoTipoObrigatórioDescrição
area_idintegerSimID da área vinculada à conexão que vai enviar a mensagem
phonestringSimNúmero de destino, só dígitos, com código do país (ex: 5511999999999)
messagestringSimTexto da mensagem a ser enviada

Exemplo de requisição

POST /api/whatsapp/send HTTP/1.1
Host: SEU-DOMINIO-ORACULO
X-Oraculo-Key: SUA_CHAVE_DE_ACESSO
Content-Type: application/json

{
  "area_id": 3,
  "phone": "5511999999999",
  "message": "Olá! Seu pedido foi confirmado."
}

Resposta de sucesso 200 OK

{
  "ok": true,
  "status": "sent",
  "chat_id": 42,
  "credit_status": {
    "balance": 123.4500,
    "depleted": false
  }
}
CampoDescrição
statusSempre "sent" em caso de sucesso — indica que a mensagem foi aceita e enviada. Não há confirmação de entrega/leitura nesta versão da API.
chat_idID interno da conversa, útil se você quiser consultar o histórico depois
credit_status.balanceSeu saldo em R$ após o débito desta mensagem
credit_status.depletedtrue quando o saldo chegou a zero ou menos após esta mensagem

Respostas de erro

CódigoSituação
400Parâmetro obrigatório faltando, área não disponível para sua chave, ou nenhuma conexão WhatsApp vinculada a essa área
401Chave de acesso ausente ou inválida
402Seus créditos acabaram
403Conta desativada, ou envio de mensagens não configurado (sem preço definido) para essa área — fale com o administrador
502Falha ao enviar — conexão WhatsApp indisponível no momento. Nada foi cobrado.

Toda resposta de erro segue o formato:

{ "error": "descrição legível do problema" }

Exemplo de uso em PHP

Usa apenas cURL, sem dependências externas:

<?php

$apiKey  = 'SUA_CHAVE_DE_ACESSO';
$baseUrl = 'https://SEU-DOMINIO-ORACULO';

/**
 * Envia uma mensagem de WhatsApp via API do Oráculo.
 * Lança RuntimeException em caso de falha (rede ou erro retornado pela API).
 */
function enviarMensagemWhatsapp(string $baseUrl, string $apiKey, int $areaId, string $telefone, string $mensagem): array
{
    $payload = json_encode([
        'area_id' => $areaId,
        'phone'   => $telefone,
        'message' => $mensagem,
    ]);

    $ch = curl_init(rtrim($baseUrl, '/') . '/api/whatsapp/send');
    curl_setopt_array($ch, [
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_POST           => true,
        CURLOPT_POSTFIELDS     => $payload,
        CURLOPT_HTTPHEADER     => [
            'Content-Type: application/json',
            'X-Oraculo-Key: ' . $apiKey,
        ],
        CURLOPT_TIMEOUT        => 30,
    ]);

    $responseBody = curl_exec($ch);
    $httpCode     = curl_getinfo($ch, CURLINFO_HTTP_CODE);
    $curlError    = curl_error($ch);
    curl_close($ch);

    if ($curlError) {
        throw new RuntimeException("Erro de conexão: {$curlError}");
    }

    $data = json_decode($responseBody, true);

    if ($httpCode !== 200) {
        $erro = $data['error'] ?? 'Erro desconhecido';
        throw new RuntimeException("Falha ao enviar mensagem ({$httpCode}): {$erro}");
    }

    return $data;
}

// Uso:
try {
    $resultado = enviarMensagemWhatsapp(
        $baseUrl,
        $apiKey,
        3,                      // area_id
        '5511999999999',       // telefone (DDI + DDD + número, só dígitos)
        'Olá! Seu pedido foi confirmado.'
    );

    echo "Mensagem enviada com sucesso. chat_id: {$resultado['chat_id']}\n";

    if (!empty($resultado['credit_status'])) {
        $saldo = number_format($resultado['credit_status']['balance'], 2, ',', '.');
        echo "Saldo restante: R$ {$saldo}\n";

        if ($resultado['credit_status']['depleted']) {
            echo "Atenção: seus créditos acabaram — recarregue para continuar enviando.\n";
        }
    }
} catch (RuntimeException $e) {
    echo "Erro ao enviar mensagem: " . $e->getMessage() . "\n";
}
Cada chamada bem-sucedida é cobrada conforme o preço por mensagem configurado para a área usada. Se você receber erro 403 dizendo que o envio não está configurado, ou 402 de créditos esgotados, entre em contato com o administrador do Oráculo.
